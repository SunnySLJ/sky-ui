# docs 目录说明

这个目录存放项目文档。

当前包含：

- `PRD.md`
  产品范围与复刻目标。
- `PLAN.md`
  实施计划与阶段拆解。
- `MIGRATION.md`
  静态稿迁移素材清单。
- `SEED.md`
  早期计划种子稿。

推荐阅读顺序：

1. `PRD.md`
   先看目标、范围和成功标准。
2. `PLAN.md`
   再看当前执行节奏、Next 架构和迁移步骤。
3. `MIGRATION.md`
   实际迁页时查静态稿来源、组件去向和素材清单。
4. `SEED.md`
   仅作早期思路参考，不作为当前主计划。

原则：

- 产品、架构、迁移、执行计划文档统一放在这里。
- 根目录尽量只保留 Next 工程本身和必要配置。
